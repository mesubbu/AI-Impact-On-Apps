# Play Store App AI Impact Visualizer

An interactive treemap visualizing the **AI impact scores** of **318 Play Store apps** across **22 categories**.

Inspired by [Karpathy's US Job Market Visualizer](https://github.com/karpathy/jobs), this tool explores how AI is reshaping the mobile app ecosystem — from productivity tools to social media, education to developer tools.

## Features

- **Squarified treemap** — apps grouped by Play Store category, with tile area proportional to group size
- **3 color layers** — toggle between:
  - **AI Impact Score** (1–10, green → red gradient)
  - **Disruption Type** (Enhancement, Automation, Replacement, AI-Native Competition)
  - **Confidence** (High / Medium / Low)
- **Stats dashboard** — histograms, tier breakdowns, and cross-charts that update per layer
- **Rich tooltips** — hover for reasoning, key drivers, confidence, and disruption type
- **Play Store linking** — click any tile to search for it on Google Play

## Quick Start

```bash
# Clone and serve
git clone <your-repo-url>
cd apps-visualizer
python3 -m http.server 8000
# Open http://localhost:8000
```

No build step. No dependencies. Just static HTML + JSON.

## Data Pipeline

The data is generated by `build_app_data.py` (one level up), which merges three audited score files:

| Source File | Apps |
|-------------|------|
| `CScore1To5.json` | Batches 1–5 |
| `CScore6789.json` | Batches 6–9 |
| `CScore10To13.json` | Batches 10–13 |

To rebuild `data.json`:

```bash
cd .. && python3 build_app_data.py
```

## Scoring Methodology

Each app is scored on a **1–10 AI Impact Scale**:

| Score | Level | Meaning |
|-------|-------|---------|
| 1–2 | Minimal | Core function has negligible AI overlap |
| 3–4 | Low | AI may improve peripheral features |
| 5–6 | Moderate | AI meaningfully enhances core workflows |
| 7–8 | High | AI fundamentally reshapes the app's value |
| 9–10 | Critical | AI-native competitors could replace this app |

**Disruption types:**
- **Enhancement** — AI adds value but doesn't threaten the app
- **Automation** — AI automates significant portions of the app's function
- **Replacement** — AI could make the app obsolete
- **AI-Native Competition** — new AI-first apps challenge this category
